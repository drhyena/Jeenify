# System Architecture

## Overview

Jeenify follows a three-tier architecture:

1.  Presentation Layer (Frontend)
2.  Application Layer (Backend)
3.  Data Layer (Database)

## Presentation Layer

Technologies: - HTML - CSS - JavaScript

Responsibilities: - User interface - Form inputs - Displaying data

## Backend Layer

Technology: - Java (Servlets / Spring Boot)

Responsibilities: - Business logic - Authentication - Data processing -
Communication with database

## Database Layer

Technology: - MySQL

Responsibilities: - Store student records - Store attendance - Store
assessment data
